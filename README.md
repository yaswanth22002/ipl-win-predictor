# IPL Win Predictor 
Real-time IPL match win probability engine using Logistic Regression.

**Live Demo**: [link here]

## Features
- Predicts win probability ball-by-ball in the 2nd innings
- Trained on 1,095 matches and 96,000+ delivery records (81.5% accuracy)
- Built with scikit-learn, Pandas, and Streamlit

## Setup
pip install -r requirements.txt
streamlit run app.py
